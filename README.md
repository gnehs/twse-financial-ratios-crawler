# 公開資訊觀測站 - 財務比率抓取
## 如何使用
```bash
# 1. 安裝 Node.js
# https://nodejs.org/zh-tw/download/
$ brew install node
# 2. 下載專案
$ git clone https://github.com/gnehs/twse-financial-ratios-crawler.git
# 3. 切換到專案資料夾
$ cd twse-financial-ratios-crawler
# 4. 安裝套件
$ npm install
# 5. 修改 config.js
# 6. 執行
$ node index.js
```
